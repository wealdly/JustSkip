[center][size=4][b]JustSkip[/b][/size]
[size=2][color=#999999]Game speed control for Crimson Desert[/color][/size][/center]
Toggle or hold a hotkey to speed up the game. Release to return to normal.
Fully configurable, supports keyboard and XInput gamepad. No Cheat Engine required.
[line]
[size=3][b]Hotkeys[/b][/size]
[b]Keyboard[/b]
[list]
[*][b]F7[/b] — 1.2x toggle [color=#999999]· subtle always-on speedup[/color]
[*][b]F8[/b] — 2x toggle [color=#999999]· travel, looting[/color]
[*][b]F9[/b] — 4x toggle [color=#999999]· cutscenes, crafting[/color]
[*][b]F10[/b] — 12x hold [color=#999999]· blast through anything[/color]
[*][b]F6[/b] — reload config
[/list]
[b]Gamepad[/b] [color=#999999](hold Back/View + button)[/color]
[list]
[*]Back + D-pad Up — 1.2x toggle
[*]Back + D-pad Left — 2x toggle
[*]Back + D-pad Down — 4x toggle
[*]Back + D-pad Right — 12x hold
[*]Back + Start — reload config
[/list]
[color=#999999]Toggles are mutually exclusive. Hold overrides any active toggle while pressed.[/color]
[line]
[size=3][b]Configuration[/b][/size]
Edit [font=Courier New]JustSkip.ini[/font] to change hotkeys, speeds, toggle/hold modes, and gamepad bindings.
Up to 6 speed slots. Press F6 to hot-reload — no restart needed.
[line]
[size=3][b]Installation[/b][/size]
[b]Requires:[/b] [url=https://github.com/ThirteenAG/Ultimate-ASI-Loader/releases]Ultimate ASI Loader[/url] (x64)
If you don't have an ASI loader yet:
[list=1]
[*]Download [b]Ultimate-ASI-Loader_x64.zip[/b] from the [url=https://github.com/ThirteenAG/Ultimate-ASI-Loader/releases/latest]latest release[/url]
[*]Rename the DLL to [font=Courier New]winmm.dll[/font]
[*]Place it in your Crimson Desert [font=Courier New]bin64/[/font] folder
[/list]
[b]Manual[/b] [color=#999999](recommended)[/color]
[list=1]
[*]Extract [font=Courier New]JustSkip.asi[/font] and [font=Courier New]JustSkip.ini[/font] into [font=Courier New]bin64/[/font]
[*]Launch the game — press F8 to verify
[/list]
[b]CDUMM[/b] [color=#999999](drag-and-drop mod manager)[/color]
[list=1]
[*]Drag the CDUMM zip onto the [url=https://www.nexusmods.com/crimsondesert/mods/207]CDUMM[/url] window
[*]Enable in the ASI Plugins tab and apply
[/list]
[b]CDMM[/b] [color=#999999](JSON Mod Manager)[/color]
[list=1]
[*]Extract the [font=Courier New]_asi[/font] folder into your CDMM [font=Courier New]mods/[/font] directory
[*]Enable and apply
[/list]
[line]
[size=3][b]How It Works[/b][/size]
Hooks [font=Courier New]QueryPerformanceCounter[/font] to scale the game's perception of time. No game code is modified — [b]survives patches[/b].
[size=3][b]Troubleshooting[/b][/size]
Set [font=Courier New]DebugLog=1[/font] in the INI. Check the generated log for "QPC hook installed".
